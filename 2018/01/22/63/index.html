YouCompleteMe的安装过程

  1. 删除原有的vim配置

    rm -rf ~/.vimrc
    rm -rf ~/.vim

  1. clone项目

    cd ~
    git clone https://github.com/wktadmin/config_files

  1. 执行安装文件

作者已经将安装过程写到了脚本里

config_files/vim/install_ubuntu.sh
  1. 进入vim执行安装插件

输入vim进入vim编辑器, 使用:PluginInstall执行安装, 等待完成.
