目的为将响应数据格式化为指定的格式 中文文档: http://www.pythondoc.com/Flask-RESTful/fields.html

示例一. 基本用法

from flask_restful import Resource, fields, marshal_with
from app.models import Cat


cat_fields = {
    'msg': fields.String,
    'status': fields.String(default='404'), # 带有默认值
    'data': fields.String(attribute='language')  # 类似别名
}


# 使用, 注意装饰器
class CatResource(Resource):
    @marshal_with(cat_fields)
    def get(self):
        return {"msg": 'hehehe', 'status':'200', 'language': 'en'}

实例二. 嵌套用法

cat_fields = {
    'id': fields.Integer,
    'c_name': fields.String
}

cats_fields = {
    'msg': fields.String(default='ok'),
    'status': fields.String(default='200'),
    'data': fields.Nested(cat_fields)  # 嵌套
}

# 使用, 注意装饰器

class CatTwoResource(Resource):
    @marshal_with(cats_fields)
    def get(self):
       cat =  Cat.query.all()
       data = {
           'msg': 'ok',
           'data': cat
       }
       return data

返回json结果为

{
    "msg": "ok",
    "status": "200",
    "data": [
        {
            "id": 1,
            "c_name": "Tom"
        },
        {
            "id": 2,
            "c_name": "Jerry"
        }
    ]
}

示例三. 嵌套url

例二中里层cat_fields添加一个字段, 更改为:

cat_fields = {
    'id': fields.Integer,
    'c_name': fields.String,
    'url': fields.Url('id')
}

并且添加一个类和url, 设置endpoint=’id’:

# 获取一只猫
class CatOneResource(Resource):
    def get(self):
        return {
            "info":'info'
        }
# 注册url
api.add_resource(CatOneResource, '/catone/', endpoint='id')

此时返回的json数据变为:

{
    "msg": "ok",
    "status": "200",
    "data": [
        {
            "id": 1,
            "c_name": "Tom",
            "url": "/catone/"
        },
        {
            "id": 2,
            "c_name": "Jerry",
            "url": "/catone/"
        }
    ]
}

默认是相对url, 为了获取指定的url格式, 可以指定'url': fields.Url('id', absolute=True

以下是废话, 一般可以跳过:

  • 指定输出相对url

    cat_fields = {

    'id': fields.Integer,
'c_name': fields.String,
'url': fields.Url('id', absolute=True)

    }
